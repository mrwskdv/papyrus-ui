# Checkbox

Input of type checkbox

## When To Use

- When the user needs to select one or more options from a list of options.

- When the user can choose between two mutually exclusive options.

## Guidelines

- Use Checkbox component within CheckboxGroup to manage checked values.

- Provide descriptive label as a child node.
