# Caption

The `Caption` component is used to display smaller descriptive text, usually accompanying images, figures, or tables.

## When To Use

- **Image Descriptions**: Use `Caption` to provide context or descriptions for images, tables, or charts.

- **Small Annotations**: Useful for adding smaller, secondary information like timestamps, notes, or disclaimers beneath
  primary content.

### Guidelines

Consider using a lighter or neutral color such as `neutral500` to ensure the caption doesn't overpower the primary text
but remains readable.
