## ImageInput

An input for uploading and previewing images with an integrated dropzone and optional multi-select.

### When to use

- **Allow quick image uploads**: Provide a simple drag-and-drop experience.
- **Show previews**: Display thumbnails to confirm selected images before submission.
- **Support multi-image workflows**: Let users add multiple images with inline remove.

### Guidelines

- **Localize labels**: Customize `editLabel`, `removeLabel`, `loadingText`, and messages.
- **Validation messages**: Use `fileInvalidFormatMessage`, `fileTooLargeMessage`, `imageTooSmallMessage`, and `tooManyFilesMessage`.
- **Accessibility**: Provide a meaningful `label` and `placeholder`. Ensure images have descriptive names via `getName` for `alt` text.
- **Controlled vs uncontrolled**: Use `value`/`onChange` for controlled forms, otherwise `defaultValue`.
