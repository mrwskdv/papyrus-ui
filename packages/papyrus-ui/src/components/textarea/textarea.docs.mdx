# Textarea

Input component for text values with multiline support.

## When To Use

- When you need to provide a text input that can span multiple lines.

### Variants

The `variant` property can be used to specify the visual variant of the input box. Available values are `primary` (default) and `secondary`.

- `primary`: Default variant with black/10 background
- `secondary`: White background with neutral border

## Guidelines

- Provide placeholder text that contains a brief description of what the input
  field is for and what type of information is expected using `placeholder`
  property.

- Use localized values for text that is accessible for the user or screen
  readers. Use `clearLabel` prop to provide custom label for the clear button.

- Use **`label`** property to provide a label for the input.

- Use **`description`**property to provide additional context or instructions for the input.

- Display helpful hints with the `message` prop or error messages with `invalid` for user guidance, especially in form
  validation scenarios.

## Accessibility

See [Accessibility Guidelines](https://developer.microsoft.com/en-us/fluentui#/controls/web/textinput#accessibility)
