# Quote

The `Quote` component is used to display block quotes, offering different text sizes and variants to fit various use cases.

## When to use

The `Quote` component is mainly used for displaying quotations.

## Guidelines

- **Readable Text Size**: Choose the appropriate size (sm, md) based on the length and emphasis of the quote.

- **Word Breaking**: Use breakWord when you have long or unbreakable content that may overflow the container.
