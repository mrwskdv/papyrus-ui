# NumericInput

Input component for collecting a number value from the user.

## When To Use

- When the user needs to enter a number value.

## Guidelines

- Provide placeholder text that contains a brief description of what the input
  field is for and what type of information is expected using `placeholder`
  property.

- Use **`label`** property to provide a label for the input.

- Use **`description`**property to provide additional context or instructions for the input.

- Display helpful hints with the `message` prop or error messages with `invalid` for user guidance, especially in form
validation scenarios.
