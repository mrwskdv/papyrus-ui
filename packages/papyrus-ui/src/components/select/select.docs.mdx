# Select

A component that allows users to select one or more options from a list of
pre-defined options.

## When to use

- When users need to select one or more options from a list of pre-defined
  options.

## Examples

### Basic

<Canvas of={Basic} />

### Multiple

The `multiple` property allows multiple values to be selected. Users will see tags displaying the selected values.

<Canvas of={Multiple} />

### Sizes

The `size` property can be used to specify the size of the input box. Available values are `sm`, `md`, and `lg`.

<Canvas of={Sizes} />

### With Icon

Use `startIcon` and `endIcon` properties to add icons to the input box.

<Canvas of={WithIcon} />

### Description

Displays description above the input. The description can be used for additional information or instructions for the
user.

<Canvas of={Description} />

### Message

Displays a hint message below the input. This message can be used as a lower-priority feedback when no validation error
has been occurred.

<Canvas of={Message} />

### Invalid

Setting the `invalid` property to `true` indicates an error or issue with the input value.

<Canvas of={Invalid} />


### Disabled

Disabled input

<Canvas of={Disabled} />

## Guidelines

- Use **`label`** property to provide a label for the input.

- Use **`description`**property to provide additional context or instructions for the input.

- Display helpful hints with the `message` prop or error messages with `invalid` for user guidance, especially in form
  validation scenarios.
